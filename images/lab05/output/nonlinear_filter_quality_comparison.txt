Сравнение медианного фильтра и фильтра усеченного среднего


Для сравнения использовалась апертура full_3x3. Медианный фильтр выбирает
среднее значение в отсортированной окрестности, а фильтр усеченного среднего
с d=4 отбрасывает два минимальных и два максимальных значения, после чего
усредняет оставшиеся пикселы.

1. Гауссов шум

Для гауссова шума лучше работает усеченное среднее. Это логично, потому что
шум распределен по всему изображению более-менее плавно, без отдельных очень
резких выбросов. После удаления крайних значений в окрестности остается
несколько пикселов, которые можно усреднить, и случайные отклонения частично
компенсируются.

Примеры:

  bnw, Gaussian var=400:
    noisy PSNR        = 23.01 dB
    median PSNR       = 25.14 dB
    trimmed mean PSNR = 25.41 dB

  imagesample, Gaussian var=625:
    noisy PSNR        = 20.31 dB
    median PSNR       = 25.79 dB
    trimmed mean PSNR = 26.60 dB

  dog, Gaussian var=625:
    noisy PSNR        = 20.21 dB
    median PSNR       = 27.38 dB
    trimmed mean PSNR = 28.40 dB

Визуальтно видно (особенно на пиксель арте), что усеченное средние вносит смешение 
(то самое сглаживание по сути, появляются новые оттенки серого, картинка размывается).
Усеченное среднее же в некотором смысле дает более четкий несмазанный результат.

2. Импульсный шум

Для импульсного шума лучше работает медианный фильтр. Тут шум представляет
собой отдельные черные и белые выбросы, и медиана хорошо их выкидывает из
результата, если в окрестности большинство пикселов нормальные.

Примеры:

  bnw, impulse p=0.05:
    noisy PSNR        = 17.25 dB
    median PSNR       = 26.69 dB
    trimmed mean PSNR = 26.34 dB

  imagesample, impulse p=0.06:
    noisy PSNR        = 17.56 dB
    median PSNR       = 30.36 dB
    trimmed mean PSNR = 29.96 dB

  dog, impulse p=0.06:
    noisy PSNR        = 17.82 dB
    median PSNR       = 37.30 dB
    trimmed mean PSNR = 36.04 dB

Визуально заметно, что для усеченного среднего выбросы полностью не убиваются усреднением, 
и некоторые "песчинки" остаются на картинке.

3. Пиксельная графика

Для pixel_art_invader результаты менее показательные. Там важны резкие
блоковые границы, а оба фильтра меняют форму объектов. Поэтому PSNR может
выглядеть странно:

  pixel_art_invader, Gaussian var=225:
    noisy PSNR        = 24.73 dB
    median PSNR       = 17.90 dB
    trimmed mean PSNR = 18.76 dB

  pixel_art_invader, impulse p=0.10:
    noisy PSNR        = 13.08 dB
    median PSNR       = 16.82 dB
    trimmed mean PSNR = 17.41 dB

Фильтры  убирают шум, но портят ступенчатые
границы. Поэтому для пиксельной графики сравнение только по PSNR не очень
хорошо отражает визуальное качество.

По визуалу тут мне больше нравится медианный фильтр, потому что он не дает лишних цветов 
(что критично для пиксельарта, там зачастую палитра узкая и фиксированная)
